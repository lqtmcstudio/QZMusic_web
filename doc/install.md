# QZ Music v2 安装与使用指南

欢迎使用 QZ Music v2！本软件采用全新的 **插件化架构**，为您提供更自由、更纯净的听歌体验。

### 📦 软件安装

1. **下载：** 请前往 **群文件** 或 **官方网站** 下载最新版的 `QZ Music v2.zip`。
* 版本说明：包含 *尝鲜版 / 测试版 / 正式版*。
* *注意：若压缩包设有密码，请查看群公告获取。*


2. **解压：** 解压下载的 ZIP 压缩包。
3. **安装：** 运行解压后的 `.apk` 文件完成安装。

---

### 🔌 插件安装

> **重要提示：** QZ Music软件内不含任何音频获取逻辑

* **尝鲜版：** 若 ZIP 内仅有 APK，则说明已临时内置测试插件。
* **测试版 / 正式版：**
1. 进入软件 **[设置]**。
2. 点击 **[插件管理]** -> **[安装插件]**。
3. 在文件选择器中选中 **ZIP 格式** 的 QZ 插件。


* *注：v2 版本不再支持旧版 v1 的 .js 格式插件。*
